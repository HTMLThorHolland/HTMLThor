Colour Key

Hey this is really important. Once you know the what our colours mean, you'll be reading through your errors like a pro. Flesh these out first before delving into the utpoic world of clean HTML...

  • Red is a syntactic error in your code or link. That's like writing a tag with a typo, or leaving out a required attribute.

  • Blue is a semantic error in your code. Semantic errors are like using lots of <br> tags instead of using <li> tags.

  • Purple are deprecated tags: tags that were once validly used in HTML but have since been 'deleted. So don't use them!
