HELP KEY

Syntax

Any error as a result of writing code incorrectly is deemed a syntax error. For example forgetting a > or ".

Semantic

An error caused by using the tags in the wrong context, or in the wrong way. For example, breaking with <br/> instead of using <li> tags.

Best practice

This error is caused when you do things in a legal but undesirable way. For example, using a table layout instead of sections for your webpage.

Deprecated tag

An error caused by using a tag or attribute that was once valid in HTML but no longer is. An example of this is the <dir> tag now replaced with <ul>.

Broken Link

An error returned when a file you have linked to in your HTML can not be found. Potentially due to a typo, or incorrect folder location.

No errors

You're a HTML hero, nothing to worry about here!

What is HTML THOR?

HTML Thor is a dedicated team of HTML Police splurging through the World Wide Web to ensure it's error free - validated to the latest HTML standard, and catering for people of all walks of life including those that use smart phones, IE, and screen readers. Today, HTML Thor presents you a validator that will take your code from a link, clipboard, a file or two, and .zip folders. HTML Thor's validator will be your best friend and learning tool so you can be certified with a HTML Hero badge every website of the way.

What is HTML5?

HTML5 is the new standard of HTML - Hypertext Markup Language... The language we use to code websites! HTML5 is jam-packed with new features, cutting back nitty gritty code bits, enhancing a website's load time, browser consistency (on execution) and screen reading compatibility. Get started with basic tutorials by checking out W3Schools' series on HTML5.

Types of Errors

Syntax Errors

A syntax error can be caused by something as simple as a typo! An example of a syntactic error in your code is like forgetting to close a tag with a > symbol or mistyping a letter, <jead> instead of <head> for example.

Semantic Errors

Semantic errors are one step above 'best practice' errors. A semantic error involves using a tag in the wrong way, for example, using <br> tags to make new lines instead of using <li> tags.

Best Practice Errors

Don't overlook these errors! Caused when your code is syntactically correct, but it's been done in a very undesirable manner, best practice errors are important to attend to for clean code.

Deprecated Tags

The deprecated tag error is caused when you use a tag or attribute that was once validly used in HTML but no longer is. An example of this is using a <strike> tag instead of using the new <del>

No Errors

Nothing to worry about here, you're a HTML Hero!

What's a .zip file?

A .zip file is a compressed version of a folder from your computer. Zipping multiple files will allow you to upload an entire website directory. find out more

Can I upload HTML 4 code?

Yes! And we recommend it! By doing so we'll be able to highlight the required HTML5 tags, any deprecated tags you've used and new best practice techniques used in HTML5.

Selecting multiple files

When uploading multiple files, hold down CTRL (or CMD for Mac) to select more than one file.

Allowed file types

You can upload files with a .html extension which have been written in any version of HTML (excluding XHTML).

Your code will be validated to HTML5 standards. HTML files containing PHP can be uploaded for validation, but what sits between your PHP tags will not be validated. CSS files can't be uploaded.
